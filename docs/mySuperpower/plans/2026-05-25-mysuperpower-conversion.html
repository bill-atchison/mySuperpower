Overview

Goal — Convert this obra/superpowers fork into a single, locally-installed Claude Code plugin named my-superpower (branded mySuperpower) whose 4 workflow skills emit HTML, while keeping the repo cleanly mergeable from upstream.
Architecture — The 4 HTML skill variants take over the canonical skill names in place. A targeted, idempotent rebrand script performs the recurring transforms (superpowers: namespace cascade + manifest name fields) so future upstream merges are one command. Branding, README, and the project CLAUDE.md are one-time edits.
Tech Stack — Git (with an upstream remote), PowerShell 7 + Bash for file ops, JSON plugin manifests, Claude Code plugin/marketplace schema. No build system; "tests" are verification commands (grep assertions, idempotence re-runs, node/python JSON parse) with expected output.

Out of scope (left untouched, like upstream's tests/): the OpenCode harness files (.opencode/, package.json's main pointer to superpowers.js) and upstream's tests/ suite.

Commit convention: every commit message ends with the trailer Co-Authored-By: Claude Opus 4.7 (1M context) <noreply@anthropic.com> (omitted from the short commands below for brevity — include it).

Create the idempotent rebrand script

Author (do not yet run) the targeted rebrand tool in both PowerShell and Bash. It performs ONLY two transforms: the superpowers: namespace cascade over skills/ + hooks/session-start, and the manifest name rewrite. It deliberately avoids a blanket "superpowers"→"my-superpower" replace, so obra/superpowers URLs and the legacy ~/.config/superpowers path are preserved.

Create: scripts/apply-my-superpower-brand.ps1, scripts/apply-my-superpower-brand.sh
  1. Write scripts/apply-my-superpower-brand.ps1 with exactly this content 
    #!/usr/bin/env pwsh
# Re-apply the my-superpower brand. Idempotent. Run on a CLEAN index only
# (never against files containing git conflict markers).
# Transform 1: namespace cascade  superpowers:<skill> -> my-superpower:<skill>
# Transform 2: manifest name       "name": "superpowers[-dev]" -> "name": "my-superpower"
$ErrorActionPreference = "Stop"
$root = Split-Path -Parent $PSScriptRoot

# --- Transform 1: namespace cascade over skills/ and the session-start hook ---
$nsTargets = @()
$nsTargets += Get-ChildItem (Join-Path $root "skills") -Recurse -File
$hook = Join-Path $root "hooks/session-start"
if (Test-Path $hook) { $nsTargets += Get-Item $hook }
foreach ($f in $nsTargets) {
  $orig = Get-Content $f.FullName -Raw
  $new  = $orig -replace 'superpowers:', 'my-superpower:'
  if ($new -ne $orig) { Set-Content $f.FullName $new -NoNewline }
}

# --- Transform 2: manifest name fields ---
$manifests = @(
  ".claude-plugin/plugin.json", ".claude-plugin/marketplace.json",
  ".codex-plugin/plugin.json", ".cursor-plugin/plugin.json",
  "gemini-extension.json", "package.json"
)
foreach ($rel in $manifests) {
  $p = Join-Path $root $rel
  if (-not (Test-Path $p)) { continue }
  $orig = Get-Content $p -Raw
  $new  = $orig -replace '"name":\s*"superpowers-dev"', '"name": "my-superpower"'
  $new  = $new  -replace '"name":\s*"superpowers"',     '"name": "my-superpower"'
  if ($new -ne $orig) { Set-Content $p $new -NoNewline }
}
Write-Host "apply-my-superpower-brand: done."
  2. Write scripts/apply-my-superpower-brand.sh with exactly this content 
    #!/usr/bin/env bash
# Re-apply the my-superpower brand. Idempotent. Run on a CLEAN index only.
set -euo pipefail
root="$(cd "$(dirname "$0")/.." && pwd)"

# Transform 1: namespace cascade over skills/ and the session-start hook
while IFS= read -r -d '' f; do
  perl -pi -e 's/superpowers:/my-superpower:/g' "$f"
done < <(find "$root/skills" -type f -print0)
[ -f "$root/hooks/session-start" ] && perl -pi -e 's/superpowers:/my-superpower:/g' "$root/hooks/session-start"

# Transform 2: manifest name fields
for rel in .claude-plugin/plugin.json .claude-plugin/marketplace.json \
           .codex-plugin/plugin.json .cursor-plugin/plugin.json \
           gemini-extension.json package.json; do
  p="$root/$rel"
  [ -f "$p" ] || continue
  perl -pi -e 's/"name":\s*"superpowers-dev"/"name": "my-superpower"/g; s/"name":\s*"superpowers"/"name": "my-superpower"/g' "$p"
done
echo "apply-my-superpower-brand: done."
